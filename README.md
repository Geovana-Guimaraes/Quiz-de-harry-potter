Quem me conhece sabe o quanto eu sou apaixonada pelo universo de Hogwarts. Por isso, decidi juntar essa paixão com o meu caminho nos estudos de tecnologia para criar o meu mais novo projeto: um Quiz de Harry Potter!

Desenvolver esse projeto do zero foi uma experiência incrível para colocar em prática meus conhecimentos em JavaScript, estruturar a lógica das perguntas, manipular elementos na tela.
